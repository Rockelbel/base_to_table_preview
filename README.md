# Base to Table Preview

飞书多维表格数据预览与 PDF 导出工具。将多维表格记录渲染为精美的表格布局，并支持一键导出 PDF。

## ✨ 功能特点

- 📊 **智能数据读取** - 自动读取飞书多维表格当前记录数据
- 🎨 **多种表格布局** - 支持标题、双列、单列、多列表格等多种布局
- 📄 **PDF 导出** - 一键导出高质量 PDF，支持 A4 纸张格式
- 🔄 **实时监听** - 自动监听记录切换，实时更新预览
- 📱 **响应式设计** - 适配不同屏幕尺寸

## 🚀 快速开始

### 环境要求

- Node.js 16+

## 📖 使用说明

### 表格布局类型

本项目支持以下几种表格布局：

| 类型 | 代码 | 说明 |
|------|------|------|
| 标题 | `4` | 大标题，居中显示 |
| 副标题 | `5` | 小字副标题，居中显示 |
| 双列 | `1` | 四个单元格，两两一组 |
| 单列 | `2` | 标签 + 内容，两列布局 |
| 多行 | `3` | 多行多列表格，支持表头 |

### 数据配置示例

```typescript
const config = [
  [
    [4, "客户信息表"],                    // 标题
    [1, "公司名称", "XX科技", "简称", "XX"], // 双列
    [2, "公司简介", "这是一家科技公司"],     // 单列
    [3,                                    // 多行表格
      ["姓名", "职位", "电话"],
      ["张三", "经理", "13800138000"]
    ]
  ]
];
```

### PDF 导出

点击页面上的「导出为PDF」按钮，即可将当前预览内容导出为 PDF 文件。

导出配置：
- 纸张：A4
- 方向：纵向
- 边距：10mm
- 分辨率：2x（高清）

## 📝 开发说明

### 本地开发

本项目基于飞书多维表格插件架构开发，需要在飞书环境中运行。

1. 在飞书多维表格中创建插件
2. 将项目 URL 配置为插件地址
3. 开启开发模式进行调试

### 自定义表格样式

可在 `index.html` 中修改 CSS 样式：

```css
/* 表格类型 1 - 双列布局 */
.table-type-1 td:nth-child(1) {
    font-weight: bold;
    width: 20%;
}

/* 自定义样式... */
```
